Spring Todo-app

- Spring 5
- Spring Boot 2
- html 5
- JUnit

- AssertJ
- Data
- Hibernate
- AOP
- Security
- Thymeleaf
- Flyaway
- Maven
- Dependency Injection
- Tomcat
